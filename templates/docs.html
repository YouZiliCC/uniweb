帮助

快速开始

  1. 注册账号 - 使用学号和邮箱注册
  2. 创建/加入工作组 - 创建新组或申请加入现有组
  3. 创建项目 - 在工作组中创建项目并配置 Docker
  4. 启动容器 - 启动容器，通过 WebShell 管理
  5. 部署应用 - 在WebShell中操作容器，部署和管理应用
  6. 访问应用 - 通过映射端口访问运行中的服务

工作组

  • • 每位用户只能属于一个工作组
  • • 组长可审批申请、管理成员、转让组长、创建新项目
  • • 组内成员共享项目管理权限

项目管理

  • • 创建项目 - 填写名称、描述，选择 Docker 镜像和端口
  • • 启动/停止 - 控制容器运行状态
  • • WebShell - 在浏览器中直接访问容器终端
  • • 访问应用 - 通过映射端口访问运行中的服务
  • • 重置容器 - 删除并重新创建容器

提示

  • • 个人、工作组、项目描述介绍文本支持 Markdown 格式，将在前端实时渲染
  • • 可在个人、工作组、项目资料页上传主题图片，将作为头像显示
  • • 点击右上角图标切换深色模式
  • • 可通过注册时输入邀请码注册为教师用户
  • • 对容器的内部端口、镜像的更改，需要重置容器以生效（重置将会删除容器内数据，请做好备份）
